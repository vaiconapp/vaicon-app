# 05 — ΕΠΟΜΕΝΑ ΒΗΜΑΤΑ

## Άμεσο επόμενο βήμα (συμφωνημένο)
**Σύγκριση vaicon-eidikes ↔ vaicon-app**: εντοπισμός λειτουργιών που υπάρχουν στις ειδικές
και λείπουν/διαφέρουν στις τυποποιημένες, ώστε ο χρήστης να αποφασίσει τι θα μεταφερθεί.
Στόχος: οι τυποποιημένες να φτάσουν/ξεπεράσουν τις ειδικές σε λειτουργικότητα.
Όλα γίνονται στο TEST — δεν αγγίζουμε την παραγωγή.

## Πώς να ξεκινήσει το νέο chat
Διάβασε ΟΛΟΝ τον φάκελο `vaicon-app/_handoff/` (αρχεία 00–06) για πλήρες context.
Μετά πρότεινε στον χρήστη τη λίστα σύγκρισης eidikes↔app και ρώτα τι θέλει πρώτο.

## Τρόπος εργασίας
- Μην κάνεις τίποτα go-live (auth παραγωγής, migration) μέχρι ο χρήστης να το ζητήσει ρητά.
- Πριν από κάθε push: version bump (αρχείο 04).
- Ζήτα άδεια πριν από μεγάλες/μη αναστρέψιμες αλλαγές.
